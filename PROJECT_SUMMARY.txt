================================================================================
TROUBLESHOOTING
================================================================================

Can't install requirements:
  - pip install --upgrade pip
  - pip install --no-cache-dir -r requirements.txt

Database connection fails:
  - Check DATABASE_URL environment variable
  - Ensure PostgreSQL running (use Docker Compose)
  - Verify credentials

API won't start:
  - Port 8000 in use? Try different port
  - Check DATABASE_URL is set

Tests failing:
  - Ensure PostgreSQL is running
  - Check database connection
  - Run pytest -v -s for detailed output

Streamlit errors:
  - Clear cache: rm -rf ~/.streamlit
  - Reinstall: pip install --upgrade streamlit
